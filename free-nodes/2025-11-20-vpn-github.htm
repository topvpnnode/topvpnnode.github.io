一、Shadowrocket：iOS平台的代理之王

Shadowrocket是一款只在iOS系统上运行的网络代理工具，支持Shadowsocks、Vmess、Trojan等多种主流协议。它不仅能实现基础的科学上网需求，还支持定制规则、流量分流、网络测速等进阶功能，是少数在稳定性与灵活性之间找到完美平衡的工具。

在Apple严苛的审核机制下，Shadowrocket依然能够常驻App Store本身就是一种“生存奇迹”。它以简约的交互界面承载复杂的功能，让非技术背景的用户也能轻松驾驭。

二、泥鳅哥是谁？他在Shadowrocket生态中的价值

“泥鳅哥”并不是开发者本人，而是一位长期活跃在科学上网圈内的技术达人。他以丰富的教程、清晰的配置指引和高质量的订阅节点被广大用户熟知。许多人第一次成功使用Shadowrocket，就是通过他提供的图文教程或推荐订阅源。

三、Shadowrocket的安装与配置步骤详解

1. 系统与下载要求

要使用Shadowrocket，设备必须满足以下基础条件：

  • 操作系统：iOS 12.0或以上

  • 设备空间：建议留有至少200MB空间用于缓存

  • 网络权限：确保App可以访问蜂窝数据或WiFi网络

你可以直接在App Store中搜索“Shadowrocket”并购买，售价通常为2.99美元（约合人民币20元左右），一劳永逸。

2. 安装后首次启动设置

  • 安装完成后打开App，会要求授权VPN设置权限。

  • 同意后，Shadowrocket会自动添加一条VPN配置至系统设置中。

  • 此时App内部仍为空，需要用户手动添加代理配置或导入订阅链接。

四、配置方式一：手动添加代理节点

如果你拥有完整的服务器信息，可以手动添加：

  1. 打开Shadowrocket，点击右上角“+”号；

  2. 选择协议类型（如Shadowsocks、Vmess、Trojan）；

  3. 填写如下信息：

    • 地址（IP或域名）

    • 端口

    • 用户ID（如UUID或密码）

    • 加密方式（如果是Shadowsocks或Trojan）

    • 路径（Path，可选）

    • TLS是否启用（依服务器配置决定）

填写完成后，保存即可。回到主界面，选中配置，点击“连接”。

五、配置方式二：使用泥鳅哥订阅链接

相比手动添加，更推荐使用“订阅”功能：

  1. 打开Shadowrocket，点击“配置”标签；

  2. 再点击“订阅”选项卡；

  3. 选择“添加订阅链接”，输入泥鳅哥发布的订阅地址；

  4. 点击“完成”并返回，再点击“更新”；

  5. 系统会自动拉取最新的节点列表；

  6. 选择任意节点，连接即可。

优势：

  • 一键导入，自动更新

  • 节点多样，质量稳定

  • 不需重复配置，省心高效

六、规则设置：按需科学上网

Shadowrocket不仅可以全局代理，也能进行精准的规则配置：

  • 全局模式：所有应用流量均通过代理（适合防火墙极严地区）；

  • 分应用代理：只对指定App走代理，如Telegram、YouTube；

  • 按域名规则：如设置*.google.com始终走代理，其余直连。

泥鳅哥通常会提供自定义规则模板供下载，大大提升用户体验。

七、使用体验优化建议

  1. 定期更新订阅：建议每周手动点击“更新订阅”，以获取新节点。

  2. 测速功能使用：长按节点可进行延迟测试，优选低Ping服务器。

  3. 选择冷门时间段连接：如清晨、深夜，避开高峰，速度更快。

  4. 开启UDP转发：部分App（如游戏）需打开“UDP转发”功能才能正常运行。

  5. 切换DNS：设置为1.1.1.1或8.8.8.8可改善连接稳定性。

FAQ

Potatso Lite 是否支持订阅更新？
Potatso Lite 支持通过订阅链接更新节点。用户在设置中输入 URL 后，应用会定期拉取最新配置文件，实现自动更新节点信息。这让日常使用更加便捷和稳定。
VLESS + XTLS 模式相比普通 VLESS 有哪些优势？
XTLS 模式减少冗余加密开销，提高传输效率，并在多连接高负载环境下表现更佳。同时仍保持 TLS 安全性，结合策略组可实现高性能和高隐蔽性的双重优势。
Matsuri 节点延迟排序功能如何使用？
Matsuri 会自动测试节点延迟，并根据结果排序。用户可以选择延迟最低的节点使用，从而优化访问速度和稳定性，尤其适合游戏、视频和海外网站加速。
SSR 的混淆和协议组合如何选择？
根据网络封锁情况选择适合的混淆和协议组合，例如 tls1.2_ticket_auth 模拟 HTTPS 流量。不同组合对抗 DPI 和封锁能力不同，需要测试后选择最稳定和高效的方案。
Clash 的规则订阅更新有哪些注意事项？
Clash 订阅更新需保证网络畅通，规则文件格式正确，并与客户端版本兼容。建议定期更新规则以获得最新节点和分流规则，同时注意避免频繁刷新造成性能问题。
Nekobox 是否支持多配置文件切换？
支持。Nekobox 允许用户创建多个配置文件并快速切换，适合在不同网络环境下使用不同规则。例如家用网络与移动数据可使用独立的配置方案。
SSR 混淆方式如何提升抗封锁能力？
混淆方式可将流量伪装成正常 HTTPS 或 HTTP 数据，降低被 DPI 检测风险。合理选择协议和混淆组合，可以提升节点稳定性和长期可用性，确保科学上网顺畅。
Hiddify Next 订阅更新后节点消失，如何恢复？
先检查订阅源是否更改格式或被下架，尝试重新粘贴订阅链接并手动刷新；若仍缺失，联系订阅提供方确认是否停用某些节点或使用了新的访问权限机制（例如 token 验证）。
V2Ray 的 ws+tls 传输模式有什么优点？
ws+tls 模式利用 WebSocket 伪装成普通 HTTPS 流量，使得防火墙难以区分代理与正常访问。该模式兼容性强，适合部署在支持 Nginx 或 Caddy 的 Web 服务器上。
Clash for Android 如何导入订阅并自动更新？
在 Clash for Android 的配置界面中选择“配置文件”→“新建配置”，粘贴订阅链接即可。启用自动更新功能后，软件会定期检查订阅源是否有新节点，并在后台自动替换旧配置，保持节点最新状态。
